# UET OOP Assignments

A centralized repository documenting my Object-Oriented Programming coursework from my second semester at UET Lahore (2025-CS-531). This portfolio showcases my C# development, bridging OOP theory with practical architecture. It features structured solutions for file I/O, 2D array matrices, and logic-driven apps, all built to industry standards.

## 📂 Directory Architecture

### `Assignment_02_PizzaPoints/`
* **Focus:** File I/O operations, string parsing, and data validation.
* **Description:** A C# console application that reads customer data from external text files to calculate and award loyalty points based on specific dynamic thresholds (N and Y).

### `Assignment_03_ProbabilityMatrix/`
* **Focus:** Multidimensional data structures and algorithmic traversal.
* **Description:** Evaluates 2D arrays to verify mathematical probability matrices, demonstrating deep array manipulation and nested loop logic.

## 🛠 Tech Stack
* **Language:** C#
* **Framework:** .NET
* **Concepts Applied:** File Handling, 2D Arrays, Encapsulation, and Algorithm Design.
